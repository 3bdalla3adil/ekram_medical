Petro Integration API Connector for Odoo v18

Effortlessly Sync Sales, Tanks, and Shifts Data

This module is a powerful integration tool designed to connect your Odoo v18 instance with the Petro Integration API (https://petrointegrationapi.cashin.sa). It automates the synchronization of sales transactions, tank measurements, and shift data, providing a unified view of your operations within Odoo.

Key Features

Sales Synchronization

Automatically fetch and store sales transactions from the Petro API, ensuring your sales data in Odoo is always up-to-date.

Tank Measurement Tracking

Keep a close eye on your tank volumes with automated measurements synchronization, aiding in better inventory and stock control.

Shift Management

Import business shifts directly into Odoo to align your workforce and operations with the data from the Petro system.

Automated & Manual Sync

Choose between fully automated scheduled actions or manual synchronization buttons for on-demand data updates.

Easy Configuration

Simply enter your API URL, Client ID, and Client Secret in the configuration dashboard to get started. No complex setup required.
